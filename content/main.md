## Proyectos actuales

### Arquitectura Web por Componentes

Desarrollo de una página web basada en componentes declarativos.

**Concepto central:**

```
CONSTRUCT.js → motor de render
data.json → estructura
componentes → renderizados recursivamente
```

Separación clara entre:
- Componentes visuales
- Mecanismos del sistema (carrito, modales, integraciones)

Objetivo: crear un sistema flexible donde la interfaz se construya mediante datos estructurados.
